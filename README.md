# Welcome to the Gemini FullStack ADK

The **Gemini FullStack Application Development Kit (ADK)** provides the tools and frameworks you need to build, test, and deploy high-quality, full-stack applications with powerful AI capabilities. Whether you're a beginner or an experienced developer, our ADK is designed to streamline your workflow and help you create amazing products.

This documentation is your guide to understanding and using the Gemini FullStack ADK. It's structured to help you find the information you need quickly and easily.

## 🚀 Get Started

If you're new here, we recommend starting with our tutorials.

*   **[Hello, World!](./tutorials/01-hello-world.md)**: A 5-minute introduction to the Gemini FullStack ADK.
*   **[End-to-End Application](./tutorials/02-end-to-end-app.md)**: Build a complete application in under 15 minutes.
